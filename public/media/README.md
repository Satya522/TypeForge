TypeForge media lives here.

Suggested structure:
- `brand/` for logos, wordmarks, favicons, app icons
- `icons/` for standalone UI icons
- `images/` for page imagery and screenshots
- `og/` for social preview banners

If you replace the logo later, keep the same filename:
- `brand/typeforge-logo-mark.svg`

That way the app will automatically pick up the new asset everywhere.
